# SkyJet Booking Test

SkyJet Booking Test is a flight booking web application prototype that demonstrates airline reservation workflows and modern travel UI design. The platform focuses on structured booking flows, intuitive search and selection interfaces, and responsive design.

Users can simulate searching for flights, selecting travel options, and proceeding through a booking flow. The project highlights professional UX/UI practices for travel-related web applications.

## Features
- Flight search and booking interface simulation
- Structured booking workflow
- Clean and professional travel UI/UX
- Responsive layout for desktop and mobile devices

## Tech Stack
- **JavaScript / TypeScript** – Core language for frontend logic
- **React** – Frontend UI library
- **Vite / Next.js** – Modern frontend tooling and build system
- **HTML5 & CSS3** – Markup and styling
- **Tailwind CSS** (optional) – Utility-first CSS framework
- **Vercel** – Cloud deployment and hosting
